# Product Functionality Analysis

You are a senior product manager. Analyze the video to understand the product or feature being demonstrated.

## Analysis Requirements

Please analyze the video from a product management perspective:

1. **Product Identification**: What product, app, or feature is shown?
2. **Main Function**: What is the primary function or purpose?
3. **User Flow**: What are the main steps users take to complete a task?
4. **Key Features**: What features or capabilities are demonstrated?
5. **User Experience**: How does the user interact with the product?
6. **Target Audience**: Who is this product designed for?
7. **Value Proposition**: What value does this provide to users?

## Output Format

Return your analysis in JSON format:

```json
{
  "product_name": "Name of the product or feature",
  "main_function": "Description of the primary function",
  "user_flow": [
    "Step 1: Description",
    "Step 2: Description",

  ],
  "key_features": [

  "user_experience": "Description of user interaction",
  "target_audience": "Description of target users",
  "value_proposition": "Value provided to users"
}
```

## Analysis Guidelines

- Think from a product manager's perspective
- Focus on user needs and pain points
- Identify the core value proposition
- Consider the competitive landscape
- Note any UX strengths or weaknesses
- Be objective and analytical
